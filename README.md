# Blockblend

Blender插件，将高分辨率模型转换为Minecraft风格的方块低模。

## 功能特性

- ✅ **Voxel Remesh转换**：使用Blender内置的高效体素化算法
- ✅ **独立立方体对象**：每个体素转换为可单独编辑的立方体
- ✅ **多种颜色模式**：原始材质、统一颜色、随机颜色、高度渐变、位置着色
- ✅ **完整UI面板**：在3D视图侧边栏提供直观的参数控制
- ✅ **撤销/重做支持**：完整的Blender undo/redo系统支持
- ✅ **性能优化**：可选的立方体合并功能
- ✅ **灵活参数**：体素大小、适应性、间隙、缩放等多项可调参数

## 安装方法

### 从源码安装

1. 将此仓库克隆到本地：
```bash
git clone https://github.com/yourusername/blockblend.git
```

2. 在Blender中安装：
   - 打开Blender
   - 进入 `Edit` → `Preferences` → `Add-ons`
   - 点击 `Install...` 按钮
   - 选择仓库中的 `__init__.py` 文件
   - 勾选 `Blockblend` 插件启用

### 从ZIP安装

1. 在GitHub上点击 `Code` → `Download ZIP`
2. 在Blender中：
   - `Edit` → `Preferences` → `Add-ons`
   - 点击 `Install...`
   - 选择下载的ZIP文件
   - 启用插件

## 使用方法

### 基础使用

1. **选择对象**：在3D视图中选择一个网格对象
2. **打开面板**：在右侧边栏找到 `Blockblend` 面板
3. **调整参数**：
   - `体素大小`：控制方块大小（值越大方块越大）
   - `创建独立立方体`：勾选以创建独立对象
   - `立方体间隙`：控制立方体之间的间隙
4. **执行转换**：点击 `转换为方块风格` 按钮
5. **撤销**：按 `Ctrl+Z` 撤销转换

### 参数说明

#### 转换设置

- **体素大小**（0.001 - 10.0）：
  - 控制方块的大小
  - 较小的值 = 更多细节，更多立方体
  - 较大的值 = 更粗糙，更少立方体

- **细节级别**（0.1 - 10.0）：
  - 自动计算体素大小的辅助参数
  - 值越大，体素越小

- **适应性**（0.0 - 1.0）：
  - `0.0`：均匀分布的方块
  - `1.0`：根据模型形状自适应细节

#### 立方体设置

- **创建独立立方体**：
  - 勾选：每个体素成为独立的立方体对象
  - 取消：保持单一网格对象

- **立方体间隙**（0.0 - 0.5）：
  - 控制立方体之间的间隙比例
  - `0.0` = 无间隙（立方体紧贴）
  - `0.5` = 较大间隙

- **立方体缩放**（0.1 - 2.0）：
  - 整体缩放立方体大小

- **合并立方体**：
  - 将所有立方体合并为单一网格对象
  - 可显著提高性能

#### 材质和颜色

- **保留材质**：
  - 将原始对象的材质应用到新立方体

- **颜色模式**：
  - `原始材质`：保留原始材质分配
  - `统一颜色`：所有立方体使用相同颜色
  - `随机颜色`：每个立方体随机颜色
  - `基于高度`：根据Z轴高度渐变着色
  - `基于位置`：根据X/Y/Z位置着色

#### 高级设置

- **平滑着色**：
  - 勾选：立方体有平滑的外观
  - 取消：平坦的方块外观（推荐用于MC风格）

- **删除原始对象**：
  - 转换后自动删除原始高模对象

## 工作流程示例

### 场景1：快速MC风格转换

```
1. 选择高模对象
2. 设置体素大小 = 0.2
3. 勾选"创建独立立方体"
4. 设置立方体间隙 = 0.05
5. 颜色模式 = "随机颜色"
6. 点击"转换为方块风格"
```

### 场景2：性能优化（大型模型）

```
1. 选择复杂模型
2. 设置体素大小 = 0.5（较大值）
3. 勾选"创建独立立方体"
4. 勾选"合并立方体"（重要！）
5. 点击"转换为方块风格"
```

### 场景3：细节保留

```
1. 选择需要细节的模型
2. 设置体素大小 = 0.05（小值）
3. 设置适应性 = 0.3
4. 颜色模式 = "基于高度"
5. 点击"转换为方块风格"
```

## 技术原理

### Voxel Remesh算法

Blockblend使用Blender内置的Voxel Remesh修饰器：
1. 将模型体素化为规则网格
2. 提取每个体素的中心点
3. 在中心点位置创建立方体primitive
4. 应用材质和颜色

### 性能考虑

- **立方体数量**：体素大小直接影响的立方体数量
  - 体素大小 0.1 → 约1000个立方体
  - 体素大小 0.05 → 约8000个立方体
  - 体素大小 0.02 → 约125000个立方体

- **合并选项**：对于大量立方体（>1000），建议启用合并
- **撤销操作**：每个转换都会占用内存，频繁转换注意内存使用

## 故障排除

### 转换失败

**错误**："对象必须是网格类型"
- **解决**：确保选择的是Mesh对象，不是曲线、灯光等

**错误**："无法提取体素中心点"
- **解决**：尝试增大体素大小，或检查模型是否有效

### 性能问题

**问题**：转换很慢或Blender卡死
- **解决**：
  1. 增大体素大小
  2. 启用"合并立方体"选项
  3. 关闭不必要的视口效果

### 视觉问题

**问题**：立方体间隙太大或太小
- **解决**：调整"立方体间隙"参数

**问题**：颜色不正确
- **解决**：检查颜色模式设置，或尝试不同的颜色模式
